フォームブリッジ（formrunなど）で、郵便番号から住所を自動入力する仕組みを作るには、以下の方法を使います：

✅ 方法：郵便番号API + JavaScript連携

ステップ1：フォームに以下のフィールドを用意

  • 郵便番号（例：zip）

  • 住所（都道府県、市区町村、番地など）

ステップ2：以下のようなJavaScriptをフォームに埋め込む

これは[郵便番号API（例えば zipcloud）]を使った例です。

html
<script>
document.addEventListener("DOMContentLoaded", function () {
  const zipInput = document.querySelector('[name="zip"]');
  zipInput.addEventListener("blur", function () {
    const zip = zipInput.value.replace("-", "");
    if (zip.length === 7) {
      fetch(`https://zipcloud.ibsnet.co.jp/api/search?zipcode=${zip}`)
        .then(response => response.json())
        .then(data => {
          if (data.results) {
            const address = data.results[0];
            document.querySelector('[name="prefecture"]').value = address.address1;
            document.querySelector('[name="city"]').value = address.address2 + address.address3;
          } else {
            alert("住所が見つかりませんでした。");
          }
        })
        .catch(() => alert("通信エラーが発生しました。"));
    }
  });
});
</script>

🔧 注意点

  • name="zip" などのフィールド名は、あなたのフォームに合わせて変更してください。

  • JavaScriptを埋め込めるかどうかは、フォームブリッジのプランや設定に依存します。

  • フォームビルダーによっては、独自の郵便番号自動入力機能がある場合もあります。

必要であれば、あなたの具体的なフォームURLや使っているサービスにあわせてコードを調整できますよ。

✅ JavaScriptのサンプルコード（formbridge用）

以下のようなコードを address2.js にして使えます：

javascript
document.addEventListener("DOMContentLoaded", function () {
  const zipInput = document.querySelector('[name="zip"]'); // 郵便番号
  const prefCityInput = document.querySelector('[name="pref_city"]'); // 都道府県・市区町村
  const streetInput = document.querySelector('[name="street"]'); // 市区町村以降

  if (zipInput) {
    zipInput.addEventListener("blur", function () {
      const zip = zipInput.value.replace("-", "");
      if (zip.length === 7) {
        fetch(`https://zipcloud.ibsnet.co.jp/api/search?zipcode=${zip}`)
          .then(response => response.json())
          .then(data => {
            if (data.results && data.results.length > 0) {
              const result = data.results[0];
              const prefCity = result.address1 + result.address2; // 東京都千代田区
              const street = result.address3; // 一番町など

              if (prefCityInput) prefCityInput.value = prefCity;
              if (streetInput) streetInput.value = street;
            } else {
              alert("該当する住所が見つかりませんでした。");
            }
          })
          .catch(() => alert("郵便番号検索に失敗しました。"));
      }
    });
  }
});

✅ 注意点

  1. [name="フィールド名"] をあなたのフォームのフィールド名に合わせて変更する必要があります
    → 例：「郵便番号」のname属性が zip ならそのままでOK

  2. フィールド名が分からない場合、formbridgeのHTMLソースを確認してください

✅ フォームブリッジ用 JavaScriptコード（郵便番号 → 住所自動入力）

javascript
document.addEventListener("DOMContentLoaded", function () {
  const zipInput = document.querySelector('[name="postal_code2"]'); // 郵便番号
  const addressInput = document.querySelector('[name="address2"]'); // 都道府県・市区町村まで
  const streetInput = document.querySelector('[name="市区町村以降_ご入居者様"]'); // 市区町村以降

  if (zipInput) {
    zipInput.addEventListener("blur", function () {
      const zip = zipInput.value.replace("-", "").trim();
      if (zip.length === 7 && /^[0-9]{7}$/.test(zip)) {
        fetch(`https://zipcloud.ibsnet.co.jp/api/search?zipcode=${zip}`)
          .then(response => response.json())
          .then(data => {
            if (data.results && data.results.length > 0) {
              const result = data.results[0];
              const address = result.address1 + result.address2; // 都道府県 + 市区町村
              const street = result.address3; // それ以降の町名など

              if (addressInput) addressInput.value = address;
              if (streetInput) streetInput.value = street;
            } else {
              alert("該当する住所が見つかりませんでした。");
            }
          })
          .catch(() => alert("郵便番号検索に失敗しました。"));
      }
    });
  }
});

  2. GitHubにアップロード → jsDelivr URL作成（前述の手順）

  3. フォームブリッジの「URL指定で追加」にそのURLを入力して有効にする

  4. フォーム画面で郵便番号を入力し、外すと自動で住所が入るようになります！
